# 📦 校园快递驿站包裹管理系统

一个基于 Python 命令行的校园快递驿站包裹管理程序，支持**库存管理员**和**取件人**双角色登录与操作。

---

## ✨ 功能特性

### 🔐 用户系统
- **库存管理员**：拥有完整的包裹管理权限
- **取件人**：查看和领取自己的包裹
- **新用户注册**：取件人可自助注册账号

### 📋 管理员功能
| 功能 | 说明 |
|------|------|
| 包裹入库 | 录入新包裹，可选关联指定取件人 |
| 取件查询 | 通过取件码快速查询并取件 |
| 显示所有包裹 | 查看当前所有包裹信息 |
| 删除超时未取包裹 | 自动清理超过指定天数的未取包裹 |
| 修改包裹状态 | 将包裹标记为已取件 |
| 删除指定包裹 | 按单号删除特定包裹 |
| 按快递公司统计 | 查看各快递公司包裹数量 |
| 按取件状态统计 | 查看已取/未取包裹数量 |

### 👤 取件人功能
| 功能 | 说明 |
|------|------|
| 查看我的包裹 | 查看关联到自己名下的所有包裹 |
| 领取包裹 | 通过取件码领取包裹 |

---

## 🛠️ 技术栈

- **语言**：Python 3
- **数据存储**：纯文本文件 (`packages.txt` / `users.txt`)
- **无第三方依赖**：仅使用 Python 标准库

---

## 🚀 运行方式

```bash
# 确保 Python 3 已安装
python main.py
```

### 默认管理员账号
| 用户名 | 密码 | 角色 |
|--------|------|------|
| `admin` | `admin123` | 库存管理员 |

> 取件人账号可通过系统内注册功能自行创建。

## 📁 项目结构

```
快递驿站管理系统/
├── main.py          # 主程序入口
├── auth.py          # 用户认证模块
├── manager.py       # 包裹管理逻辑
├── package.py       # 包裹数据模型
├── web_app.py       # Web 服务接口（扩展功能）
├── templates/
│   └── index.html   # Web 页面模板
├── packages.txt     # 包裹数据文件
├── users.txt        # 用户数据文件
└── README.md        # 本说明文档
```

---

## 📝 数据文件格式

### `packages.txt`
```
单号,收件人姓名,取件码,快递公司,入库时间,是否已取件(0/1),关联取件人用户名
```

### `users.txt`
```
用户名,密码,角色(admin/recipient),显示名称
```
